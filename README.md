## Repository Structure
Here is the skeleton of the current repository
```bash
├── docs/
│   ├── part1/          # Part1 report
│   │   ├── Report - Part 1.pdf
│   │   ├── domain model.jpg
│   │   ├── use-case-diagram.jpg
│   ├── part2/          # Part2 report
│   │   ├── Report - Part 2.pdf
│   │   ├── class diagram/
│   │   ├── database diagram/
│   │   ├── sequence diagrams/
│   ├── part3/          # Part3 report

│   │   ├── sequence diagrams/
```
